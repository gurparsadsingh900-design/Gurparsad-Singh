##⚙️ Getting Started

OpenCV (cv2)
NumPy

##📂 Installation
1. Clone the repository:
   ```bash
   git clone
    cd smart-vision-productivity-suite
    ```
2. Install Python dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Installation
    npm install
    npm start
