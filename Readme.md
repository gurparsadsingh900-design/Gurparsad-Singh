AI-powered applications for eye care, image processing, and productivity automation.

## 📌 Description

This project is a collection of intelligent tools focused on **computer vision, automation, and productivity**.

## 💡 Why This Project?

Students spend 10–15 hours on screens, causing serious eye strain.
This project uses AI to monitor eye activity and help prevent long-term damage.

## 📂 Installation

```bash
git clone https://github.com/gurparsadsingh900-design/smart-vision-productivity-suite.git
cd smart-vision-productivity-suite
pip install -r requirements.txt
npm install
```
